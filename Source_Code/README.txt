-- HOW TO IMPORT THE JAVA PROJECTS INTO YOUR WOKSPACE --
Click right onto your projects area select Import > Existing projects into your worskpace > Select the folders of the Projects > Finish

-- HOW TO EXPORT THE JAVA PORJECTS MODIFIED TO THE DATA INTEGRATION (KETTLE/PENTAHO) --
In Eclipse select:
File > Export > JAR File > Select the resources to export > finish
Export into [Data Integration Folder]/plugins/steps and enjoy it :p
